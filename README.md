# 🔍 Python Port Scanner

A simple but powerful port scanner built with Python.
Scans a target host for open ports and saves results to a file.

---

## 📌 What is a Port Scanner?

A port scanner checks which ports on a target server are open.
Open ports mean services are running — useful in cybersecurity
for identifying potential vulnerabilities.

---

## 🚀 Features

- Scans any target domain or IP address
- Accepts custom port ranges
- Clearly displays open ports
- Automatically saves results to a timestamped file

---

## 🛠️ Requirements

- Python 3.x
- No external libraries needed (uses built-in socket module)

---

## ▶️ How to Run

```bash
python scanner.py
```

Then follow the prompts:

Enter target (e.g. google.com): google.com
Enter start port: 1
Enter end port: 100
